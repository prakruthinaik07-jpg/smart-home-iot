# smart-home-iot
Smart Home Automation using Raspberry Pi Pico W, Sensors and MicroPython

🏠 Smart Home Automation using Raspberry Pi Pico W

📌 Project Description

This project is an IoT-based Smart Home system using Raspberry Pi Pico W and MicroPython.
It monitors environmental conditions and controls devices automatically and manually.

---

⚙️ Features

- 🔥 Smoke detection alert
- 🚶 Motion detection (security)
- 🌡 Temperature-based fan control
- 💡 Light control using LDR
- 📱 Manual control via mobile
- 🔔 Alert notifications

---

🧰 Components Used

- Raspberry Pi Pico W
- DHT11 Sensor
- PIR Sensor
- MQ-2 Smoke Sensor
- LDR Sensor
- Relay Module
- Buzzer
- LEDs

---

🧠 Working Principle

- When motion is detected → Light turns ON
- When temperature increases → Fan turns ON
- When smoke detected → Alert + Buzzer
- When darkness detected → Light turns ON

---

💻 Technologies

- MicroPython
- MQTT (Adafruit IO)
- Embedded Systems
- Telegram Notification
